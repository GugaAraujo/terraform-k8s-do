# Terraform Challenge

For this challenge, we need a script that automates infrastructure management,<br>
provisioning a WordPress application and a MySQL database.

The entire flow must be run only once.

## Setup

```bash
# init terraform
$ terraform init

# create a plan for changes
$ terraform plan

# apply changes
$ terraform apply

# remove infra
$ terraform destroy
```
